English version : 

Nothing is to be modified except for the variables in the variables.txt file.

If you change the name of a file, make sure to update the name in the makefile.

The makefile allows to compile all the necessary .cpp files of the program. Just run the makefile once the variables are saved.

The results are outputted in the form of .txt files, so you need to use software like gnuplot or matplotlib, for example, to visualize the results.

Version francaise : 

Le programme a pour objectif de modéliser le modèle d'Ising. 

Rien n'est à modifier en dehors des variables dans le fichier variables.txt.

Si vous changer le nom d'un fichier, faites attention à bien changer le nom dans le makefile. 

Ce dernier permet de lancer tout les .cpp du programme nécéssaires. Il suffit juste de lancer le makefile une fois les variables enregistrées. 

Les résultats sortent sous forme de .txt, il faut donc utiliser gnuplot ou matplotlib par exemple pour afficher les résultats.
